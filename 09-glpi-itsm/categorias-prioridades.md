# Categorias e prioridades de chamados

## Categorias

| Categoria | Exemplos |
|---|---|
| Hardware | computador não liga, monitor sem imagem, teclado com falha |
| Software | aplicativo não abre, erro em sistema, instalação de programa |
| Rede | sem internet, DNS, Wi-Fi, cabo de rede |
| Acesso | senha expirada, usuário bloqueado, permissão |
| Impressora | offline, fila travada, driver |
| VPN | não conecta, erro de autenticação, instabilidade |
| Microsoft 365 | Outlook, Teams, OneDrive, Office |

## Prioridade

| Prioridade | Critério |
|---|---|
| Baixa | Afeta um usuário, sem impacto crítico |
| Média | Afeta atividade importante de um usuário ou setor |
| Alta | Afeta vários usuários ou operação importante |
| Crítica | Paralisação geral ou impacto severo no negócio |

## Observação

Prioridade deve considerar impacto e urgência, não apenas a percepção do usuário.
