# Evidências

Esta pasta pode ser usada para armazenar evidências fictícias ou prints de laboratório, como:

- saída de comandos de rede;
- prints de inventário;
- exemplos de chamados;
- testes de scripts PowerShell;
- relatórios de diagnóstico.

## Importante

Não inserir dados pessoais, senhas, informações reais de empresas ou imagens que exponham terceiros.
