# Checklist de atendimento Help Desk N1

## 1. Identificação do usuário

- Nome do usuário
- Área/departamento
- Telefone ou e-mail para retorno
- Patrimônio ou nome do computador
- Local físico ou trabalho remoto

## 2. Entendimento do problema

- Qual problema está ocorrendo?
- Quando começou?
- O problema ocorre sempre ou às vezes?
- Aparece alguma mensagem de erro?
- O problema afeta apenas um usuário ou vários?
- Houve alguma alteração recente no computador, senha, sistema ou rede?

## 3. Classificação inicial

- Hardware
- Software
- Rede / Internet
- Impressora / periférico
- Acesso / senha
- E-mail / Microsoft 365
- VPN
- Sistema interno

## 4. Testes básicos

- Reiniciar aplicativo
- Reiniciar computador, quando aplicável
- Verificar conexão de rede
- Confirmar usuário e senha
- Verificar mensagens de erro
- Testar com outro navegador ou perfil
- Validar se outros usuários têm o mesmo problema

## 5. Registro do chamado

- Descrição clara do problema
- Evidências, se houver
- Passos já realizados
- Impacto no usuário
- Prioridade
- Status

## 6. Encerramento

- Confirmar com o usuário se o problema foi resolvido
- Registrar a solução aplicada
- Informar orientações preventivas
- Encerrar o chamado com descrição objetiva
