# Guia básico de instalação de software

## Antes da instalação

- Confirmar se o software é autorizado pela empresa.
- Verificar compatibilidade com o Windows.
- Confirmar se exige licença.
- Baixar apenas de fonte oficial ou repositório autorizado.
- Verificar se o usuário tem permissão para instalação.

## Durante a instalação

- Executar como administrador quando necessário.
- Seguir o padrão definido pela empresa.
- Evitar instalar componentes desnecessários.
- Registrar versão instalada.

## Após a instalação

- Abrir o software e testar funcionamento.
- Confirmar atalhos e permissões.
- Validar com o usuário.
- Registrar no chamado.
- Atualizar inventário, se aplicável.
