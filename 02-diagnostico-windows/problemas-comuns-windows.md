# Problemas comuns no Windows e abordagem inicial

## Computador lento

Possíveis causas:

- muitos programas iniciando com o Windows;
- pouco espaço em disco;
- excesso de arquivos temporários;
- uso alto de CPU ou memória;
- atualizações pendentes;
- malware ou software indesejado.

Procedimentos:

1. Verificar Gerenciador de Tarefas.
2. Verificar espaço em disco.
3. Reiniciar o equipamento.
4. Verificar programas de inicialização.
5. Conferir atualizações.
6. Registrar evidências.

## Aplicativo não abre

Procedimentos:

1. Verificar mensagem de erro.
2. Reiniciar o aplicativo.
3. Reiniciar o computador.
4. Verificar permissões do usuário.
5. Reinstalar ou reparar o aplicativo, se autorizado.
6. Escalonar se houver erro sistêmico.

## Sem acesso à internet

Procedimentos:

1. Verificar cabo ou Wi-Fi.
2. Testar `ipconfig`.
3. Testar `ping 8.8.8.8`.
4. Testar `ping google.com`.
5. Verificar DNS.
6. Reiniciar adaptador de rede.
7. Escalonar para infraestrutura se for falha geral.
