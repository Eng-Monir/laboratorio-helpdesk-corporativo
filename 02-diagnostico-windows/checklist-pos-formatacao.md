# Checklist pós-formatação de computador

## Sistema operacional

- Windows instalado corretamente
- Nome do computador configurado conforme padrão
- Usuário local ou corporativo configurado
- Atualizações do Windows aplicadas

## Drivers

- Driver de rede
- Driver de vídeo
- Driver de áudio
- Driver de chipset
- Driver de impressora, se necessário

## Programas básicos

- Navegador
- Pacote Office ou Microsoft 365
- Leitor de PDF
- Antivírus ou solução corporativa
- Ferramenta de acesso remoto, se autorizada
- Sistemas internos da empresa

## Configurações

- Data e hora corretas
- Idioma e teclado corretos
- Rede Wi-Fi ou cabo testado
- Impressora configurada, se necessário
- VPN configurada, se aplicável

## Validação

- Testar internet
- Testar login
- Testar Office
- Testar impressão
- Confirmar com usuário
- Registrar entrega do equipamento
