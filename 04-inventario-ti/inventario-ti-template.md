# Modelo de inventário de ativos de TI

Campos recomendados:

| Campo | Descrição |
|---|---|
| Patrimônio | Código do equipamento |
| Tipo | Desktop, notebook, monitor, impressora etc. |
| Marca | Fabricante |
| Modelo | Modelo do equipamento |
| Número de série | Serial number |
| Usuário responsável | Nome do usuário |
| Setor | Departamento |
| Localização | Local físico |
| Sistema operacional | Windows 10, Windows 11 etc. |
| Processador | Modelo do processador |
| Memória RAM | Quantidade de RAM |
| Armazenamento | HD/SSD e capacidade |
| Status | Em uso, estoque, manutenção, descartado |
| Data de entrega | Data de entrega ao usuário |
| Observações | Informações adicionais |

## Boas práticas

- Manter informações atualizadas.
- Registrar transferências de equipamento.
- Registrar equipamentos em manutenção.
- Evitar dados pessoais desnecessários.
- Controlar acessórios como carregadores, mouses e teclados.
