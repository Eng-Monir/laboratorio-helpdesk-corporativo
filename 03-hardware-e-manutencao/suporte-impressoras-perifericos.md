# Suporte a impressoras e periféricos

## Impressora offline

Procedimentos:

1. Verificar se está ligada.
2. Verificar cabo USB ou conexão de rede.
3. Conferir papel e toner/cartucho.
4. Verificar fila de impressão.
5. Reiniciar spooler de impressão.
6. Reinstalar driver, se necessário.
7. Registrar no chamado.

## Comando para reiniciar spooler

```powershell
Restart-Service Spooler
```

## Teclado ou mouse não funciona

Procedimentos:

1. Verificar conexão USB ou bateria.
2. Testar em outra porta USB.
3. Reiniciar computador.
4. Testar outro periférico.
5. Substituir equipamento, se confirmado defeito.

## Monitor sem imagem

Procedimentos:

1. Verificar cabo de energia.
2. Verificar cabo de vídeo.
3. Verificar entrada correta do monitor.
4. Testar outro cabo ou monitor.
5. Verificar se o computador liga normalmente.
