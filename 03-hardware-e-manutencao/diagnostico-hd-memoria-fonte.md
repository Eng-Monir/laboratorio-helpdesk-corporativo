# Diagnóstico básico de HD, memória e fonte

## HD / SSD

Sinais de possível problema:

- lentidão extrema;
- erros ao abrir arquivos;
- travamentos frequentes;
- ruídos em HD mecânico;
- sistema demorando muito para iniciar.

Ações iniciais:

- verificar espaço livre;
- observar mensagens de erro;
- registrar comportamento;
- solicitar backup antes de intervenção;
- encaminhar para análise avançada, se necessário.

## Memória RAM

Sinais de possível problema:

- tela azul;
- reinicializações inesperadas;
- travamentos aleatórios;
- aplicativos fechando sozinhos.

Ações iniciais:

- verificar uso de memória no Gerenciador de Tarefas;
- testar com poucos aplicativos abertos;
- registrar padrão de falha;
- encaminhar para teste físico, se necessário.

## Fonte de alimentação

Sinais de possível problema:

- computador não liga;
- desliga sozinho;
- reinicia sem motivo aparente;
- cheiro de queimado;
- falha ao energizar componentes.

Ações iniciais:

- verificar cabo de energia;
- testar tomada/filtro de linha;
- observar LEDs e ruídos;
- não abrir fonte sem capacitação adequada;
- encaminhar para manutenção autorizada.
