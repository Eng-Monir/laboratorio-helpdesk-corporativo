# Estudos introdutórios: Microsoft 365 e Active Directory

## Microsoft 365

Ferramentas comuns no ambiente corporativo:

- Outlook
- Teams
- OneDrive
- Word
- Excel
- PowerPoint
- SharePoint

Chamados comuns:

- usuário não consegue acessar e-mail;
- Teams não abre;
- OneDrive não sincroniza;
- licença não atribuída;
- senha expirada;
- dificuldade de acesso em novo dispositivo.

## Active Directory

O Active Directory é usado para gerenciar usuários, computadores, grupos e permissões em ambientes corporativos.

Conceitos básicos:

- usuário;
- grupo;
- unidade organizacional;
- domínio;
- política de grupo;
- reset de senha;
- bloqueio e desbloqueio de conta.
