# Troubleshooting de conectividade

## Cenário 1: Sem internet apenas em um computador

1. Verificar cabo ou Wi-Fi.
2. Testar outro site.
3. Executar `ipconfig`.
4. Validar gateway e DNS.
5. Testar `ping 8.8.8.8`.
6. Testar `ping google.com`.
7. Reiniciar adaptador de rede.
8. Registrar evidências.

## Cenário 2: Internet funciona, mas sistema interno não abre

1. Testar outros sites.
2. Confirmar endereço do sistema.
3. Testar em outro navegador.
4. Limpar cache do navegador.
5. Verificar se outros usuários têm o mesmo problema.
6. Escalar para equipe responsável pelo sistema, se necessário.

## Cenário 3: Wi-Fi conecta, mas não navega

1. Esquecer rede e conectar novamente.
2. Verificar IP recebido.
3. Reiniciar adaptador Wi-Fi.
4. Testar DNS.
5. Verificar se outros dispositivos navegam.
