# Laboratório de Help Desk Corporativo

Portfólio prático de **Help Desk** e **Suporte Técnico** com foco em atendimento a usuários, diagnóstico de problemas em Windows, hardware, software, redes básicas, VPN, inventário de ativos, segurança básica e documentação de chamados.

Este projeto foi criado para demonstrar organização, raciocínio técnico e capacidade de documentação em rotinas de suporte corporativo.

## Objetivo

Demonstrar conhecimentos práticos iniciais em suporte técnico, incluindo:

- atendimento a usuários;
- abertura, classificação e acompanhamento de chamados;
- diagnóstico básico de hardware e software;
- suporte em Windows;
- comandos básicos de rede;
- documentação técnica;
- inventário de ativos de TI;
- manutenção preventiva;
- suporte a VPN;
- boas práticas básicas de segurança da informação;
- scripts simples em PowerShell.

## Ferramentas e tecnologias abordadas

- Windows 10/11
- PowerShell
- Prompt de Comando
- Pacote Office
- Conceitos básicos de redes
- VPN
- Inventário de ativos
- Help Desk / Service Desk
- GLPI / ferramentas de chamados
- Microsoft 365 e Active Directory, em nível introdutório
- Documentação técnica

## Observação

Este repositório é um laboratório de estudos e simulações. Os chamados e evidências são fictícios, criados para demonstrar procedimentos técnicos de suporte.
